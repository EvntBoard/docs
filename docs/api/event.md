# All events available

## Application

### app-load

Fires when application start

Property :
- id: __string__
- event: __string__
- emittedAt: __Date__
- emitter: __null__
- payload: __null__


### click

Fires when someone click on a button

Property :
- id: __string__
- event: __string__
- emittedAt: __Date__
- emitter: __string__
- payload:
    - idButton: __string__
    - idTrigger: __string__
